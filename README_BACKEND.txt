INSTRUCTIONS D'INSTALLATION DU BACKEND
======================================

L'application mobile ne peut pas fonctionner car elle n'arrive pas à contacter le serveur backend.
Le dossier backend n'a pas été trouvé dans votre installation XAMPP.

Veuillez suivre ces étapes IMPÉRATIVES :

1. Ouvrez l'explorateur de fichiers.
2. Allez dans le dossier de votre projet : c:\Users\moham\devmob
3. Copiez le dossier nommé "backend".
4. Allez dans le dossier d'installation de XAMPP, généralement : C:\xampp\htdocs
5. Créez un nouveau dossier nommé "devmob" (s'il n'existe pas).
6. Collez le dossier "backend" à l'intérieur de "devmob".

Chemin final attendu :
C:\xampp\htdocs\devmob\backend

Vérification :
Vous devez avoir des fichiers comme "get_products.php" dans :
C:\xampp\htdocs\devmob\backend\get_products.php

7. Assurez-vous que XAMPP est lancé (Apache et MySQL doivent être verts).
8. Redémarrez l'application mobile (secouez le téléphone -> Reload ou tapez "r" dans le terminal Metro).
